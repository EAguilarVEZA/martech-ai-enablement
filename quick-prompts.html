Prompts you can paste, by app

Each card: the moves that matter, plus one copy-ready prompt. Swap the bracketed parts for your work.

OutlookEmail & calendar

  • Summarize a long thread into decisions, questions, and action items.
  • Draft or rewrite a reply — set the tone and length.
  • Coach my writing; find a time and draft the invite.
Copy this prompt Catch me up on my inbox from the last 2 days. Group by topic, flag what needs a reply today, note decisions I was asked to make, and draft short replies for the top 3.

TeamsMeetings & chat

  • Intelligent recap: decisions and action items with owners.
  • "What did I miss?" for a call or a busy channel.
  • Draft the follow-up message and next-steps list.
Copy this prompt Recap this meeting: key decisions in 3-5 bullets, action items grouped by owner with due dates, anything assigned to me, and a short follow-up I can post to the channel.

WordDocuments

  • Draft from a prompt, outline, or a few bullets.
  • Rewrite for clarity, tone, length, or audience.
  • Summarize long docs; reference files with "/".
Copy this prompt Turn these notes into a one-page campaign brief with objective, audience, key message, channels, timeline, owners, and approvals. Keep it executive-friendly. [paste notes]

ExcelData & analysis

  • Write and explain formulas in plain English.
  • Surface trends, outliers, and summaries.
  • Suggest charts, PivotTables, conditional formatting.
Copy this prompt Analyze this campaign performance table: summarize the main trend, flag outliers, suggest one chart and PivotTable, and give me the month-over-month growth formula with an explanation.

PowerPointPresentations

  • Create a deck from a Word doc or a prompt.
  • Generate speaker notes and slide summaries.
  • "Review this presentation" for structure fixes.
Copy this prompt Create a 7-slide leadership update from this brief. Per slide: one takeaway headline, 2-3 bullets, a suggested visual, and speaker notes. End with the questions I should be ready to answer.

Copilot ChatAcross your work

  • Reason over mail, files, meetings, chats, and people.
  • Summarize or compare files without opening them.
  • Build no-code agents for repeatable tasks.
Copy this prompt Using my recent emails, files, meetings, and chats about [project], give me current status in 3 bullets, top risks, what needs my decision this week, and link the sources you used.

Make any prompt better G · C · S · F

The four-part habit that lifts every Copilot answer. Miss one and the output drifts.

1

Goal

Say the outcome — a reply, a readout, a 7-slide deck.

2

Context

Name the audience and tone: "for leadership," "brief and warm."

3

Source

Attach the file, select the table, or reference with "/".

4

Format

Ask for headings, a table, bullets, or a word count — then refine.
