Role-based AI playbooks

The program should meet each teammate where their work actually happens. These tracks show how Copilot, Claude, Cowork, Claude Code, and the OH MarTech AI Challenge can support each discipline.

Website + Builder.io

Digital experience builders

Use AI to review landing pages, create content models, QA components, draft accessibility checks, summarize Builder.io page changes, and generate test plans before release.

  • Starter project: landing page QA copilot.
  • Claude Code use: inspect frontend code, tracking, schemas, and tests.
  • Proof: before/after page review with risks and fixes.
Salesforce CRM apps

CRM application builders

Use AI to translate business requests into user stories, field requirements, validation rules, flow logic, release notes, admin documentation, and UAT scripts.

  • Starter project: CRM intake-to-user-story generator.
  • Claude use: pressure-test requirements and edge cases.
  • Proof: one CRM feature brief plus UAT checklist.
Marketing Cloud + email

Email strategy and journey teams

Use AI for audience/message maps, journey QA, subject line variants, personalization logic review, send readiness, segmentation questions, and post-send learnings.

  • Starter project: email journey readiness reviewer.
  • Copilot use: summarize related emails, meetings, and approvals.
  • Proof: campaign QA checklist and learning summary.
Data architecture

Data and reporting architects

Use AI to explain data lineage, draft metric definitions, document dashboards, identify data quality risks, create stakeholder-friendly summaries, and propose validation checks.

  • Starter project: metric definition library.
  • Claude use: turn technical data notes into executive explanations.
  • Proof: one dashboard data dictionary entry.
Physician relations CRM

PRM CRM and reporting support

Use AI to summarize CRM activity patterns, draft reporting narratives, identify follow-up gaps, create field adoption support, and translate stakeholder asks into report specs.

  • Starter project: physician relations report readout template.
  • Copilot use: meeting recap and follow-up extraction.
  • Proof: reporting brief with assumptions and caveats.
Content strategy

Content and campaign strategists

Use AI to repurpose approved source material, map audience needs, create editorial briefs, check tone, draft FAQs, and align copy across email, web, paid, and internal updates.

  • Starter project: approved-source content repurposing studio.
  • Claude use: build content matrices and message variants.
  • Proof: one source-to-channel content map.
Team leadership

Manager and program lead

Use AI to manage intake, prioritize experiments, summarize progress, identify training gaps, review risk, and keep University, challenge, and adoption metrics visible.

  • Starter project: AI adoption dashboard and office hours agenda.
  • Claude use: synthesize Q&A, progress, and challenge ideas.
  • Proof: monthly enablement summary.
Shared martech ops

Cross-functional launch support

Use AI to align web, CRM, email, content, analytics, and leadership around one campaign launch checklist, shared risks, and handoff owners.

  • Starter project: integrated launch command center.
  • Tools: Copilot for Microsoft context, Claude for synthesis, Code for technical checks.
  • Proof: one launch plan with owners and verification.
