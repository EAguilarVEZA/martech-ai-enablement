Microsoft Copilot first

Because Orlando Health is a Microsoft shop, teach the team to start with Copilot for everyday Microsoft 365 work: email, meetings, documents, spreadsheets, presentations, files, and quick workday catch-up. Use Claude when the task needs deeper reasoning, long-form synthesis, reusable artifacts, or technical build support.

Best first step

Use Copilot where the work already lives

If the work is in Outlook, Teams, Word, Excel, PowerPoint, OneDrive, or SharePoint, start there so Copilot can use the immediate context.

Efficiency

Reduce daily friction

Summarize email threads, draft replies, prep meeting agendas, clean notes, analyze spreadsheets, and turn rough ideas into slides.

License awareness

Features vary by access

Copilot experience depends on your Microsoft 365 subscription, work account, admin settings, and whether a Microsoft 365 Copilot add-on license is assigned.

When to switch

Bring Claude in for deeper work

Use Claude for broader campaign thinking, long-context synthesis, prompt libraries, artifacts, agent-style reviews, and Claude Code technical projects.

Where to find Copilot

Microsoft 365 apps vary by web/desktop version and by license, but the pattern is similar: look for the Copilot button in the ribbon, top app area, side pane, or contextual prompt area. These mockups are orientation guides, not exact screenshots.

Copilot daily starter

Workday catch-up

Help me catch up on my workday.

Focus on:
- Important emails and Teams messages
- Meetings I need to prepare for
- Decisions or blockers related to [campaign/project]
- Follow-up items where I am the owner

Return:
1. Top priorities
2. Draft replies or next actions
3. Meetings that need prep
4. Anything I should escalate
Use this at the start of the day in Microsoft 365 Copilot.
Copilot spreadsheet helper

Campaign data readout

Analyze this campaign performance data.

Please:
- Summarize the main trend
- Identify outliers or data quality concerns
- Suggest useful filters, charts, or PivotTables
- Explain any formula you recommend
- Create a short executive summary for marketing leadership

Do not invent missing data. Flag anything that needs verification.
Use this in Excel after formatting data as a table or supported range.
Copilot meeting prep

Agenda and pre-read

Prepare me for my next meeting about [project/campaign].

Use available context from my calendar, emails, chats, and documents if available.

Create:
- Meeting objective
- Suggested agenda
- Key background points
- Decisions needed
- Questions I should ask
- Follow-up template
Use this in Outlook, Teams, or Microsoft 365 Copilot Chat.
Copilot to Claude handoff

Move from efficiency to strategy

Create a clean summary I can use in Claude for deeper strategy work.

Include:
- Goal
- Audience
- Current facts
- Decisions already made
- Open questions
- Source documents or meetings used
- Risks and assumptions
- What I need Claude to help with next

Keep it concise and remove sensitive details that are not needed.
Use when Copilot helps gather context, then Claude helps think deeper.
